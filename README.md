# MouseReader

MouseReader é um script Python simples que monitora e exibe a posição atual do mouse (coordenadas X e Y) em tempo real no terminal.

## Funcionalidades

- Exibe continuamente as coordenadas X e Y do mouse.
- Atualiza a posição a cada 0.1 segundos.
- Pode ser interrompido pressionando Ctrl+C.

## Dependências

- Python 3.x
- Biblioteca `pyautogui`

Instale as dependências com:

```
pip install pyautogui
```

## Como executar

1. Navegue até o diretório do projeto.
2. Execute o script:

```
python mouse.py
```

3. Pressione Ctrl+C para encerrar.

## Arquivos

- `mouse.py`: Script principal.
- `mouserdr.ico` e `mouserdr.png`: Ícones do projeto.
- `output/`: Diretório para arquivos de saída (ex.: executáveis compilados).
